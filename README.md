# Stock Analyst Dashboard

A Python tool that pulls live stock market data and returns the a moving averages graph, a correlation heatmap, and a percent return of the stock in a graph.

## Features
- Percent Returns on stocks
- Moving averages (50 or 200 days)
- Correlation Heatmap
- User Input
- Invalid ticker detection

## Technologies Used
- Python
- yfinance API
- pandas
- plotly.express

## How to Use
1. Clone the repository
2. Install dependencies: `pip install yfinance`, `pip install pandas`, `pip install plotly`
3. Run `python analysis.py`
